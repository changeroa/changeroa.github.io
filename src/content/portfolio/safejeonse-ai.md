# SafeJeonse AI (안심전세 AI)

An agentic AI system that investigates real estate documents to detect potential rental fraud risks using a ReAct-based detective architecture.

## Features
- **Fraud Detection**: Analyzes contracts and registry documents for risk factors.
- **ReAct Agents**: Uses a detective-persona agent to "investigate" properties.
- **OCR Integration**: Extracts data from physical document scans.

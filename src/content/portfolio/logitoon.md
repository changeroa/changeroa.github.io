# Logitoon - AI Story Director

AI-powered children's picture book generator that transforms complex topics into engaging 12-panel visual stories.

## Features
- **Generative Storytelling**: Uses Gemini API to create consistent narratives.
- **Visual consistency**: Maintains character consistency across panels.
- **Interactive Editor**: Allows users to refine text and regenerate images.

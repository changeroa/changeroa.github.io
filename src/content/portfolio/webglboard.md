# webGLBoard (IdealLM)

A high-performance AI context preparation tool featuring a custom WebGL-based canvas for structuring and managing data inputs for AI models.

## Features
- **High Performance**: WebGL rendering for large datasets.
- **Visual Context**: Organize AI context graphically.
- **Modern Stack**: Built with React 19 and Vite.

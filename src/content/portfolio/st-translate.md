# Code-Switching ASR & Translation

A real-time pipeline for recognizing and translating Korean-English code-switching speech using Whisper LoRA and MarianMT.

## Features
- **Fine-tuned Whisper**: Optimized for mixed language speech (Code-switching).
- **Real-time Processing**: Streamlit interface for live demonstration.
- **Translation Pipeline**: Seamlessly translates transcribed text.
